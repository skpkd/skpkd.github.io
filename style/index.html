Первый большой заголовок заметки

На этой странице собраны все стили оформления и гайды построения страниц.

На данный момент существую два вида построения страниц: с выравниванием по левому краю и по центру. Этот текст и примеры ниже набраны в колонке по левому краю. В начале страницы распологается блок с картинкой и заголовком. Конечно, это не обязательный элемент, вполне можно использовать просто заголовок H1.

У этого абзаца есть выноска. Ему добавляется класс with-side, с относительным выравниванием. Следом идет блок side, с абсолютным. На узком экране выноска перелазит в основную колонку.

Текст выноски наследует цвет родительского блока. Первая строка выравнивается по базовой линии первой строки (я надеюсь, что это так), дальше ритм повторяется через строку.
Тут есть выноска слева

Вот у этого абзаца тоже есть выноска в виде картинки. Картинка в выноске занимает 100% ширины, но не больше 100% своего реального размера. Дальше идет бесполезный текст для придания объема. Со временем я вынес большинство элементов в отельные cscc для того, чтобы управляться с базовым набором было легче. Разово используемы селекторы лежат в файле custom. В var лежат все переменные, ими удобно управлять.
